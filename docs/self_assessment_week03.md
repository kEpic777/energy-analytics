## Weakest area

My weakest area is independent implementation.

I understand many explanations when they are shown to me, but I still struggle to produce the code or SQL from scratch without help. This means I need more rehearsal days where I redo previous work without copying.

## Focus for improvement

For the next stage, I should focus on:

1. Rewriting old SQL queries from memory.
2. Explaining each query in plain English.
3. Practicing Git branch workflow more than once.
4. Debugging small errors without immediately asking for help.
5. Building one small analysis from start to finish every few days.
